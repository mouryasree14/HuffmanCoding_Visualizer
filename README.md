# Huffman Coding Visualizer

A simple and interactive web application to understand the Huffman Coding algorithm through visualization.

## Features

- Enter custom text input
- Generate Huffman Tree
- View character frequencies
- See Huffman binary codes
- Encode and decode text
- View compression statistics
- Responsive design

## Technologies Used

- HTML
- CSS
- JavaScript

## How to Use

1. Enter any text.
2. Click Generate Tree.
3. View frequencies and Huffman codes.
4. Encode or decode the text.

## Purpose

This project helps students and beginners learn Huffman Coding in an easy and visual way.
